hic_matrix
==========

Description
-----------

A track for matrix files. Only cool format and h5 format from `HiCExplorer <https://hicexplorer.readthedocs.io/en/latest/index.html>`_ are supported. For other format, please first convert to a supported format, for example with `hicConvertFormat <https://hicexplorer.readthedocs.io/en/latest/content/tools/hicConvertFormat.html#hicconvertformat>`.
